# Known Limitations

This project is playable but not finished.

## Gameplay and rules surfaces

- Cube pass/drop after a human double is not fully implemented.
- Beaver handling is not exposed as a proper UI path.
- Real opponent-strength selection is not wired into GNUbg evaluation settings yet.
- SGF save/load UI is not present.
- Tutor mode and analysis UI are not present.
- Hints/equity/MWC surfaces remain future work.

## UI and UX

- The current start screen is intentionally provisional.
- Board design is usable but not final.
- Accessibility, landscape/portrait decisions, and small-screen polish need dedicated work.
- Some convenience gestures are deliberately conservative and do nothing when ambiguous.

## Platform and release engineering

- The current focus is debug/development builds.
- Release signing, Play Store readiness, and full Android compatibility testing are not complete.
- Wider real-device testing is still required.

## Documentation

- The LaTeX master document remains the detailed engineering record.
- Public-facing GitHub documentation is being split into README plus shorter docs to make the project easier to understand.
